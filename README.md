# 静态分析基础教程

> 基于南京大学软件分析课程。

教程网址：https://static-analysis.cuijiacai.com/

## 前置要求

作为先决条件，您需要了解数据结构和算法，并熟悉一种编程语言（Java 会比其他语言更好，因为我们是以它为例进行讲解的）。编译原理不是先决条件，尽管理解它会更好。

该课程面向高年级本科生和各级研究生。
